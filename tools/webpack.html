动态打包所有依赖项，创建所谓的依赖图
      此工具用于在命令行中运行 webpack
    1. 创建一个目录 > 初始化 npm (npm init -y) > 在本地安装 webpack，接着安装 webpack-cli（npm install --save-dev webpack webpack-cli）;
    2. package.json中的main去掉，加私有，防止意外发布你的代码
        +   "private": true,
        -   "main": "index.js",
    2. 安装js中的依赖lodash   npm install --save lodash
    3. 打包npx webpack
